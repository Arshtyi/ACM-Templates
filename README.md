# 项目介绍

-   ACM 板子
-   包括 Markdown（使用 Typora 主题 [Drake-Light](https://github.com/liangjingkanji/DrakeTyporaTheme)) 与 LaTeX(XeLaTeX^2)两种格式
-   Script 目录下是一个用于复制粘贴的脚本，将所有 PDF 复制到/PDFs 目录下
-   Releases 采用 Github Actions 自动构建和发布
